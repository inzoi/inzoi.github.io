《定性化学分析的性能和不确定度评估》软件使用说明书 制作人：黄挺 日期：2024.7 1. 软件简介 1.1 软件的简介和描述： 本软件是根据“EURACHEM指南”，针对定性化学分析进行性能与不确定度评估的软件： 《EURACHEM/CITAC指南：定性化学分析的性能和不确定度评估》(2021) EURACHEM/CITAC Guide Assessment of performance and uncertainty in qualitative chemical analysis (2021) 1.2 软件工作流程： 首先在“模式”菜单选择一种模式，然后在输入文本框中输入数据； 可以在“设置”菜单中选择普通或科学计数法、以及小数位数； 最后选择“计算”菜单，软件就会在输出文本框计算结果。 1.3 技术特点： 针对EURACHEM指南中参数多，计算复杂等情况，可以迅速计算出所有评价与不确定度等结果。 对输入的数据都进行了验证。如果输出结果是不可计算的会自动输出“Not available”。 可以选择普通计数法或科学计数法、以及不同的小数位数，方便进一步计算。 2. 软件功能 2.1 模式1：列联表 输入参数的符号、名称以及要求： tp：真阳性案例数，必须是非负整数。 fp：假阳性案例数，必须是非负整数。 fn：假阴性案例数，必须是非负整数。 tn：真阴性案例数，必须是非负整数。 P(+)：阳性案例的先验概率，必须是非负实数。 P(-)：阴性案例的先验概率，必须是非负实数，P(+)+P(-)必须等于1。 输出参数的符号、名称以及要求： p：阳性结果数，p=tp+fp。 n：阴性结果数，n=fn+tn。 pc：阳性案例数，pc=tp+fn。 nc：阴性案例数，nc=fp+tn。 TP：真阳性率，即SS（灵敏度），TP=tp/pc。 FP：假阳性率，FP=fp/nc。 TN：真阴性率，即SP（特异度），TN=SP=tn/nc。 FN：假阴性率，FN=fn/pc。 PPV：“阳性预测值”或“精密度”，PPV=tp/p。 NPV：“阴性预测值”，NPV=tn/n。 E：效率，E=(tp+tn)/(p+n)。 Y：尤登指数，Y=SS(%)+SP(%)-100。 LR(+)：阳性结果的似然率，LR(+)=TP/FP。 LR(-)：阴性结果的似然率，LR(-)=TN/FN。 威尔逊区间A1：A1=2tp+1.96^2。 威尔逊区间A2：A2=1.96*(1.96^2+4tp*fn/(tp+fn))^0.5。 威尔逊区间A3：A3=2(tp+fn+1.96^2)。 LL(SS,95)：SS在95%置信区间的低限，LL(SS,95)=(A1-A2)/A3*100%。 HL(SS,95)：SS在95%置信区间的高限，HL(SS,95)=(A1+A2)/A3*100%。 威尔逊区间B1：B1=2tn+1.96^2。 威尔逊区间B2：B2=1.96*(1.96^2+4fp*tn/(fp+tn))^0.5。 威尔逊区间B3：B3=2(fp+tn+1.96^2)。 LL(SP,95)：SP在95%置信区间的低限，LL(SP,95)=(B1-B2)/B3*100%。 HL(SP,95)：SP在95%置信区间的高限，L(SP,95)=(B1+B2)/B3*100%。 PP：阳性的后验概率，PP=(P(+)/P(-)*LR(+)/[(P(+)/P(-)*LR(+)+1] PN：阴性的后验概率，PN=(P(-)/P(+)*LR(-)/[(P(-)/P(+)*LR(-)+1] 2.2 模式2：最大限量 输入参数的符号、名称以及要求： c(max)：最大容许浓度c(max)。 c：测量浓度c。 u(c)：测量浓度的标准不确定度u(c)。 输出参数的符号、名称以及要求： 如果合格，输出Conforming, c <= c(max)，和PP （阳性的后验概率，NORM.DIST(c(max), c, u(c), TRUE)） 如果不合格，输出Non-conforming, c > c(max)，和PN （阴性的后验概率，1-NORM.DIST(c(max), c, u(c), TRUE)） 2.3 模式3：最小限量 输入参数的符号、名称以及要求： c(min)：最小容许浓度c(min)。 c：测量浓度c。 u(c)：测量浓度的标准不确定度u(c)。 输出参数的符号、名称以及要求： 如果合格，输出Conforming, c >= c(min)，和PP （阳性的后验概率，1-NORM.DIST(c(min), c, u(c), TRUE)） 如果不合格，输出Non-conforming, c < c(min)，和PN （阴性的后验概率，NORM.DIST(c(min), c, u(c), TRUE)） 2.4 模式4：区间限量 输入参数的符号、名称以及要求： c(min)：最小容许浓度c(min)。 c(max)：最大容许浓度c(max)。 c：测量浓度c。 u(c)：测量浓度的标准不确定度u(c)。 输出参数的符号、名称以及要求： 如果合格，输出Conforming, c(min) <= c <= c(max)，和PP （阳性的后验概率，NORM.DIST(c(max),c,u(c),TRUE)-NORM.DIST(c(min),c,u(c),TRUE)） 如果不合格，输出Non-conforming,cc(max)，和PN （阴性的后验概率，1-NORM.DIST(c(max),c,u(c),TRUE)+NORM.DIST(c(min),c,u(c),TRUE)）
